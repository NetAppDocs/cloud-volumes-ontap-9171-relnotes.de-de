= Seitentitel kommt hierher
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Verwenden Sie diese Vorlage für Artikel, Themen, wie auch immer Sie sie nennen. Die Datei index.adoc enthält nicht die korrekten Schlüsselwörter für eine Artikelseite und ihre Verwendung als Vorlage kann später zu Veröffentlichungsproblemen führen.


NOTE: Löschen Sie diese Datei vor der ersten Veröffentlichung, da die Datei auch dann veröffentlicht wird, wenn sie nicht mit der Seitenleiste verknüpft ist. Nachdem Sie einige Artikel erstellt haben, können Sie diese stattdessen als Ausgangspunkt verwenden.

Wenn sich Ihre Artikel in einem Inhaltsunterordner befinden, müssen Sie zwei Anpassungen an der oben gezeigten Notation vornehmen:

* `:imagesdir:../media/`
* `permalink: folder-name/sample-article.html`
